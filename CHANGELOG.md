### Added

- Classic Minesweeper game logic with first-click safety, flood-fill reveal,

### Changed

- Panning is constrained to the board edges; when the whole board fits on
  screen, panning is disabled so a stray drag can't move it.
- Reworked the toolbar: a centred new-game button (an SF Symbol restart icon
  tinted by state ŌĆö neutral while playing, green on win, red on loss) replaces
  the emoji face, with the mode toggle and grouped scores/settings controls
  flanking it, all consistent SF Symbols.
- The toolbar holds up on very narrow widths: counters shrink instead of
  pushing the timer out, and the difficulty picker's label is hidden (it
  wrapped). macOS window minimum lowered to 320├Ś420.
- **Navigation restructure.** Game configuration moved to a single **New Game

  Home** + the flag toggle, with a left/right **handedness** setting). The

### Fixed

- The board now follows system light/dark changes on iOS/iPadOS (previously a
  System ŌåÆ Dark ŌåÆ System toggle could leave the grid stuck dark).
- Restoring a saved game now ignores out-of-bounds coordinates and recomputes the
  cleared-cell count from the board, so a corrupt or tampered save can't produce a
  broken or unwinnable game.
